# Аналітичний пайплайн у Docker Workspace

Даний проєкт демонструє контейнеризацію багатомодульного застосунку для аналізу даних. Система автоматично завантажує дані, перевіряє їх якість, проводить статистичне дослідження та візуалізує результати.

## Структура проєкту
- **db**: База даних PostgreSQL 15.
- **data_load**: Завантаження `dataset.csv` у БД.
- **data_quality_analysis**: Перевірка на дублікати та пропуски.
- **data_research**: Обчислення базових статистик.
- **visualization**: Побудова графіків розподілу та розсіювання (PNG).
- **web**: Flask-інтерфейс для відображення результатів.

## Технічні особливості
- **Docker Compose**: Оркестрація всіх 6 контейнерів.
- **Docker Volumes**: Спільне сховище для передачі звітів та графіків між сервісами.
- **Docker Network**: Ізольована мережа для безпечної взаємодії модулів.

## Як запустити

1. Переконайтеся, що **Docker Desktop** запущений.
2. Помістіть ваш вхідний файл у папку `dataset/dataset.csv`.
3. Відкрийте термінал у корені проєкту та виконайте:
   ```bash
   docker compose up --build
   ```
4. Після завершення роботи аналітичних модулів (статус exited with code 0), відкрийте браузер:
http://localhost:5000
